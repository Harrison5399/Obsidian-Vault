
`{{}}` - represents a bag
`{}` - represents a set

- `COUNT({a, b, c}) = 3`
- `COUNT({{a,c,c,b,b,b}}) = 6`
- `SUM({1,4,7}) = 12`
- `SUM({{1,1,1,1,1}}) = 5`
- `AVG({1,4,6}) = 4`
- `AVG({{1,1,1,1,1}})=1`
- `MIN({1,4,7}) = MIN({{1,1,1,1,1}}) = 1`
- `MAX({1,4,7}) = 7`
- `MIN({John,Eric,Ann}) = Ann` - by the lengths of the strings
